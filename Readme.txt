Cara menjalankan Program Cipher Kriptografi

1. Pastikan Python sudah terinstall dengan numpy
2. Download file program_kriptografi.py
3. Buka file dengan Visual Studio Code (atau program lainnya)
4. Jalankan program seperti biasa
5. Jika tampilan antarmuka sudah muncul, pilih cipher yang ingin dicoba
6. Masukkan plainteks dan kuncinya
7. Pilih opsi enkripsi atau dekripsi, dan hasilnya akan muncul dibawah
